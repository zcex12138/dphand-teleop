# Dphand-teleop
This repository is for teleoperation and simulation visualization of the DPhand dexterous hand.

## Install
```
pip install -e .
```
## Run (with `test=True`)
```
python visualize_mojoco.py
```

### Franka + DPhand Teleoperation
```
python visualize_franka_dphand_teleop.py
```
